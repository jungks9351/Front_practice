# Movie App

### 목적

Movie App을 통한 연습

- Build Environment : Webpack, Babel
- language : TypeScript
- Framework : React.js
- State Manage : Redux, Redux Saga
- Style : Styled-components

### 기능

[TMDB](https://www.themoviedb.org/documentation/api?language=ko) 에서 제공하는 OPEN API 사용

- 영화 인기 순위 TOP 20
- 영화 검색기능
